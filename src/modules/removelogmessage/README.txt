Remove Log Message by Moses Blumenstiel (moses at flamingvan dot com)

Hide the log text field or the whole revision fieldset by role and on any content type.

INSTALLATION
------------

1. Copy the files to your sites/SITENAME/modules directory.

2. Enable the  module at Administer >>
   Site building >> Modules (admin/build/modules).
   
USE
---   

Configure the module at  Administer >> Site configuration >> Remove Log Message (admin/settings/removelogmessage)
